# Gemini-AI-Assistant
A lightweight, interactive AI chatbot UI built using HTML, CSS, and JavaScript, integrated with Google's Gemini API. This project allows users to chat with an AI model in a clean and responsive interface with light/dark theme toggle functionality.

🚀 Features
💬 Chat interface with auto-scroll and typing area

🎨 Gradient text styling for "Gemini" branding

🌗 Dark/Light mode toggle

⚡ Smooth message animation

🔄 Real-time response loading indicator

🧠 Connects to Google Gemini API for AI responses

🛠️ Technologies Used
🔹 HTML5 for structure

🔹 CSS3 for design and dark theme

🔹 Vanilla JavaScript for interactivity and API handling

Google Gemini API (Flash model)



🧪 How to Run

Clone the repo:

git clone https://github.com/your-username/gemini-ai-assistant.git

cd gemini-ai-assistant

Add your Gemini API key in script.js:

const APIKey = "YOUR_API_KEY_HERE";

Open index.html in your browser (no server needed).

⚙️ How It Works
    When a user submits a prompt, the input is added to the chat area.
    A loading message is shown while waiting for the API.
    On response, the message is appended with styling.
    The page auto-scrolls up after each message, keeping UI clean.
    Dark mode is toggled using the ☀️/🌙 icon.
